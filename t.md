 Промпт для фронтенд-разработчика (Vercel)

## Задача
Создать красивый одностраничный сайт, где пользователь вводит своё имя и получает персональную, искреннюю похвалу от ИИ.

---

## Бэкенд API

**Базовый URL:** `https://recycling-register-contacted-mirror.trycloudflare.com/`

**Заголовок авторизации:** каждый запрос должен содержать заголовок `X-API-Key` с секретным ключом.

CORS включён, запросы с любого домена (в том числе с Vercel) проходят.

### `POST /praise`

Генерирует уникальную похвалу по имени.

**Запрос:**
```json
POST /praise
Headers:
  Content-Type: application/json
  X-API-Key: super-secret-key-123

Body:
{
  "name": "Анна"
}
```

**Ответ (200):**
```json
{
  "praise": "Анна, ты невероятный человек! Твоя доброта и искренность делают этот мир светлее..."
}
```

**Ошибки:**
- **403** — отсутствует или неверный заголовок `X-API-Key`.
- **422** — `name` пустое или не прошло валидацию.
- **Сетевая ошибка** — бэкенд недоступен.

---

## Требования к интерфейсу

1. **Поле ввода имени**
   - Placeholder: "Введите ваше имя"
   - Валидация: минимум 1 символ, максимум 100
   - Автофокус при загрузке страницы

2. **Кнопка "Получить похвалу"**
   - Дизейблится, если имя пустое
   - Показывать loader/spinner во время ожидания ответа
   - Предотвращать двойной клик (debounce)

3. **Блок с похвалой**
   - Плавная анимация появления (fade-in / typewriter)
   - Приятный шрифт, хорошие отступы
   - Эмодзи или иконки для настроения ✨

4. **Обработка ошибок**
   - Если сеть упала — показать дружелюбное сообщение "Попробуйте позже"
   - Если имя не прошло валидацию — подсветить поле красным

5. **Адаптивность**
   - Mobile-first (320px+)
   - Desktop: красиво по центру экрана

6. **Дополнительно (по желанию)**
   - Кнопка "Скопировать похвалу"
   - Кнопка "Поделиться" (Web Share API)
   - Фон с градиентом / анимированными частицами
   - Лёгкие звуковые эффекты при получении похвалы

## Пример кода (fetch)

```javascript
async function getPraise(name) {
  const res = await fetch("https://christmas-msg-determining-docs.trycloudflare.com/praise", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": "super-secret-key-123",
    },
    body: JSON.stringify({ name }),
  });

  if (!res.ok) {
    throw new Error("Ошибка сервера");
  }

  const data = await res.json();
  return data.praise;
}
```
